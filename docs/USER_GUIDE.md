- **运营商端**: https://mrgun.chu-jiao.com/operator

## 一、运营商账户

- 交易记录查看

- 运营点列表查看

### 应用授权
- 查看已授权应用列表
- 申请新应用授权
- 查看授权申请状态

### 使用数据
- 游戏使用记录查询
- 按运营点统计
- 按应用统计
- 按时间统计（日/周/月）
- 玩家数量分布统计
- 数据导出

### 消息中心
- 查看系统通知
- 标记消息已读
- 历史消息查询

## 系统特性

- **实时计费**: 按玩家数量实时扣费
- **安全认证**: JWT Token + API Key双重认证
- **数据统计**: 多维度可视化报表
- **审计追踪**: 所有操作记录审计日志
- **消息通知**: 余额不足、价格调整自动提醒
- **移动友好**: 响应式设计支持移动设备访问
